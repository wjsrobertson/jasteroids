Jasteroids - Asteroids in JavaScript

AKA kiss my asteroid

Keys:
  • N: New game
  • S: Toggle sound on / off. No sound in Internet Explorer because fuck Internet Explorer
  • A: Toggle apeshit mode on / off
  • SPACE: shoot
  • ARROW KEYS: move
