Jasteroids - Asteroids in JavaScript

AKA kiss my asteroid

Controls:
  • F: toggle fullscreen
  • N / gamepad: new game
  • SPACE / gamepad: shoot
  • ARROW KEYS / gamepad: move
  • S: toggle sound on / off
  • A: toggle apeshit mode on / off (warning - flashing lights)

Please note - Internet explorer does not support gamepads / fullscreen mode / sound terribly well. Please play this game in Chrome or Firefox for best results.
